# 娘の誕生日写真 インフィニットズームWebアプリ 要件定義・実装計画書

## 1. プロジェクト概要
娘の誕生日に毎年撮影している写真（前年の写真を額縁に入れて持っている写真）を活かし、過去から現在へと写真がシームレスに切り替わる（縮小・回転・ズームイン等）インフィニットズーム演出を楽しめるWebアプリケーションを開発する。

## 2. 目的
- 娘の成長記録を、単なるスライドショーではなく、視覚的に魅力的で感動的な体験として振り返ることができるようにする。

## 3. 機能要件
- **写真のシームレスなトランジション:**
  - 画面の縮小、拡大、回転エフェクトを用いて、現在の写真の額縁部分から次の写真（あるいは前の写真）へとシームレスに遷移する演出（ドロステ効果）。
- **再生コントロール:**
  - 自動再生機能。
  - 手動（スワイプ、スクロール、またはボタンクリック）による任意の年への移動機能。
- **年数・メッセージ表示:**
  - 写真の切り替わりに合わせて、その写真の撮影年や年齢、短いメッセージをオーバーレイ表示する。
- **画像のプリロード:**
  - アニメーションを滑らかにし、読み込み待ちによるカクツキを防ぐため、画像の事前読み込みを行う。

## 4. 非機能要件
- **パフォーマンス:**
  - 高解像度の画像を複数枚扱うため、ハードウェアアクセラレーションを効かせたCSS Transformの使用や、表示領域外の画像のDOMからの切り離しを行う。
  - 60fpsでの滑らかなアニメーションを目指す。
- **レスポンシブデザイン:**
  - スマートフォン、タブレット、PCのそれぞれの画面サイズで最適に表示されること。
- **使用技術:**
  - 言語: TypeScript
  - UIフレームワーク: React
  - アニメーション: Vanilla CSS Transform (requestAnimationFrameで制御) または GSAP / Framer Motion
  - ビルドツール: Vite

## 5. 前提条件・制約事項
- 写真データは `img/` ディレクトリ内に配置され、ファイル名等で順序が特定できること。
- 各写真における「額縁の座標情報（四隅の位置、傾きなど）」が、メタデータ（JSON等）として事前に定義されていること。これにより正確なズームと回転機能を実現する。

## 6. 実装フェーズの提案
1. **フェーズ1: プロトタイプ作成**
   - 2〜3枚のダミー画像を用意し、手動で座標と回転を指定して、ズーム・回転がシームレスに繋がるかPoC（概念検証）を行う。
2. **フェーズ2: 座標取得補助ツールの開発（推奨）**
   - 実際の写真上で額縁の4隅をクリックすることで、プログラムで利用するJSON配列（座標情報）を生成する簡易なヘルパー画面を作成。これにより数十枚の写真データの登録を自動化・簡略化する。
3. **フェーズ3: 本実装とUIの作成**
   - Reactアプリケーションとしてコンポーネント化し、実際の写真データを組み込む。さらに年数やメッセージを表示するUIコンポーネントを作成する。
4. **フェーズ4: アニメーションのブラッシュアップとデプロイ**
   - イージングの調整やパフォーマンス改善を行い、Webに公開する。
